# Dashcam Music Video Project

Automatically turns daily dashcam footage into artistic, beat-synced music videos.

## Structure

- `src/` — core Python code
- `data/raw/` — dashcam AVI files from DCIMA and DIMCB
- `output/` — rendered MP4 outputs

## Setup
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
